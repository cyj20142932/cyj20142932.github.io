Vue源码探析——Diff算法的鼻祖snabbdom！

snabbdom是一个著名的虚拟DOM库，是Diff算法的鼻祖，Vue的源码也是借鉴了snabbdom的！本文简单介绍如何搭建snabbdom测试环境！

Github仓库

https://github.com/snabbdom/snabbdom

snabbdom是一个使用Typescript编写的库，因此需要了解部分Typescript的知识！Github上虽然不提供编译好的js版本的库，但是我们可以在npm上安装js版本

npm i -S snabbdom

初始化项目学习snabbdom

npm init
npm i -S snabbdom

snabbdom库目录结构

搭建snabbdom的测试环境

我们使用webpack和webpack-dev-server来搭建snabbdom的测试环境比较简单,使用webpack 5的版本

npm i -D webpack@5 webpack-cli@3 webpack-dev-server@3

webpack.config.js配置

module.exports = {
  entry: "./src/index.js",
  output: {
    publicPath: "xuni",
    filename: "bundle.js",
  },
  devServer: {
    port: 8080,
    contentBase: "public",
  },
};

package.json文件

"scripts": {
    "dev": "webpack-dev-server"
  },

index.html文件

<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title></title>
</head>

<body>

    <!--虚拟路径-->
    <script src="/xuni/bundle.js"></script>
</body>

</html>

然后运行

npm run dev

这样简单的snabbdom测试环境就搭建好了！

官方示例

import { init } from "snabbdom/init";
import { classModule } from "snabbdom/modules/class";
import { propsModule } from "snabbdom/modules/props";
import { styleModule } from "snabbdom/modules/style";
import { eventListenersModule } from "snabbdom/modules/eventlisteners";
import { h } from "snabbdom/h"; // helper function for creating vnodes

const patch = init([
  // Init patch function with chosen modules
  classModule, // makes it easy to toggle classes
  propsModule, // for setting properties on DOM elements
  styleModule, // handles styling on elements with support for animations
  eventListenersModule, // attaches event listeners
]);

const container = document.getElementById("container");

const vnode = h(
  "div#container.two.classes",
  {
    on: {
      click: () => {
        alert("haha");
      },
    },
  },
  [
    h("span", { style: { fontWeight: "bold" } }, "This is bold"),
    " and this is just normal text",
    h("a", { props: { href: "/foo" } }, "I'll take you places!"),
  ]
);
// Patch into empty DOM element – this modifies the DOM as a side effect
patch(container, vnode);

const newVnode = h(
  "div#container.two.classes",
  {
    on: {
      click: () => {
        alert("hehe");
      },
    },
  },
  [
    h(
      "span",
      { style: { fontWeight: "normal", fontStyle: "italic" } },
      "This is now italic type"
    ),
    " and this is still just normal text",
    h("a", { props: { href: "/bar" } }, "I'll take you places!"),
  ]
);
patch(vnode, newVnode); // Snabbdom efficiently updates the old view to the new state
